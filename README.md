<h1>The Cozy Diner</h1>
<p>The Cozy Diner is a 50s diner-inspired restaurant menu website created for the Codédex HTML final project. It features a small menu with burgers, fries, and milkshakes, along with an order form where users can choose quantities, milkshake flavors, add-ons, and special requests for a more customizable order.</p>

<h2>Features</h2>
<ul>
  <li>50s diner-inspired theme</li>
  <li>Sticky navigation header</li>
  <li>Animated typewriter effect</li>
  <li>Hover animations for buttons and menu cards</li>
  <li>Order form with number inputs, radio buttons, checkboxes, and a textarea</li>
  <li>Footer with social links</li>
  <li>Basic responsive design</li>
</ul>
